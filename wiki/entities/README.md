# wiki/entities

这里放对象页。

对象页负责承接那些“具体而持续重要的东西”，比如：

- 人
- 项目
- 产品
- 组织
- 账号
- 长期关注对象

它回答的不是“这个词是什么意思”，而是：

> 这个对象现在是什么、为什么重要、最近发生了什么、和什么强相关。

它和别的页型的区别：

- 不是概念页：概念页讲最小定义，对象页讲当前状态
- 不是主题页：主题页围绕一个问题组织认知，对象页围绕一个具体对象组织信息
- 不是目标页：目标页讲推进方向和阶段，对象页讲对象本身目前怎样

什么时候适合建对象页：

- 同一个人 / 项目 / 产品反复出现在不同来源里
- 你已经不止一次想更新“它现在到底怎样”
- 它和多个目标、来源或主题开始形成稳定关系

当前这里先作为正式落点预留层，不急着一次把所有旧内容都迁过来。更稳的做法，是先拿真实材料跑样板，再逐步补页。
