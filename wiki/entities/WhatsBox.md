# WhatsBox

## 这是什么

`WhatsBox` 是 rollout 历史聊天归档里一条相对独立的 app 项目线。和 `WeBox` 相比，它的会话量更小，但问题簇更集中，主要围着首页分享 / 邀请、VIP / 账号状态、活动反馈和多语言适配在推进。

## 为什么它现在重要

因为后面如果要从总归档里回查这类增长入口、活动弹窗、会员状态相关问题，先有一个对象入口会比一直翻总归档省力得多。

## 当前状态

- 它现在已经有了一份项目级聊天来源切片：[[Codex rollout - WhatsBox 项目切片（2026-02-26 至 2026-04-17）]]。
- 当前最集中的问题簇，是首页分享 / 邀请活动、奖励回写刷新、VIP / 账号状态 UI、路由弹窗、事件埋点和多语言适配。
- 目前还没有围绕 `WhatsBox` 的正式目标页、主题页或方法页；它当前更适合作为对象入口先接住原始语境。

## 关键关系

- 它的母语境来源是 [[Codex rollout 历史聊天归档（2026-02 至 2026-04）]]。
- 直接服务它的项目级来源页是 [[Codex rollout - WhatsBox 项目切片（2026-02-26 至 2026-04-17）]]。
- 它和 [[WeBox]] 一样，都是从总归档里先切出来、再决定是否继续细分的 app 项目对象。

## 最近变化

- 2026-04-29：从总归档里切出第一份 `WhatsBox` 项目级聊天子归档，并补出对应来源页。

## 当前未决问题

- 是否还要继续把 `WhatsBox` 细分成“分享 / 邀请线”和“VIP / 账号状态线”这类更小问题簇。
- 是否值得基于更多正式来源再补 `WhatsBox` 的目标页或更稳定的知识页。
