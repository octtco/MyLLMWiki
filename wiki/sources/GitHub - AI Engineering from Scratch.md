# GitHub - AI Engineering from Scratch

## 一句话总结

AI Engineering from Scratch 是一个强调“学、做、发给别人用”的 AI 工程课程 / 手册样本，适合补机器学习从学习到交付的工程化入口。

## 当前范围

本页只整理 README 入口，不拆 435 lessons，也不评估课程质量。

## 摘要

- README 以 `Learn it. Build it. Ship it for others.` 组织项目定位。
- 它把课程拆成多个 phase 和 lesson，并强调每节课要产出可交付物。
- README 提到 built-in agent skills，可兼容 Claude、Cursor、Codex、OpenClaw、Hermes 等。
- 对 [[机器学习工程入口]] 来说，它补的是从“先看数据”之后继续走向工程交付的学习路线样本。

## 证据或原文线索

- GitHub API 抓取时显示 MIT license，stars 约 16.3k，仓库未归档。
- README 标注 `435 lessons`、`20 phases`，并包含 `The shape of the curriculum`、`Every lesson ships something`。

## 当前可支持的判断

- 当前来源支持：AI 工程学习可以按课程、产物和交付链条组织，而不只是模型概念堆叠。
- 当前可保守迁出：学习型仓库也可以作为工程入口样本，但需要后续抽样检查内容质量。

## 当前边界

- 课程体量很大，本页不判断每节课是否准确。
- 不把它写成唯一学习路线。
