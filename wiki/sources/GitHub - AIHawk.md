# GitHub - AIHawk

## 一句话总结

AIHawk 是一个自动化求职投递的 AI web agent 项目，README 说明其核心架构开源，但因版权原因移除了第三方 provider plugins。

## 当前范围

本页只整理 README 和 GitHub API 元数据；未运行投递流程，未接入招聘平台，也未复核各平台条款和自动投递风险。

## 摘要

- README 把 AIHawk 定位为 Jobs Applier AI Web Agent。
- README 说明项目曾被多家媒体报道，并强调当前仓库保留核心架构但移除第三方 provider plugins。
- GitHub API 抓取时显示 Python 仓库，AGPL-3.0 license，stars 约 29868，仓库当前 archived。

## 当前可支持的判断

- 当前来源支持：AIHawk 是求职流程自动化 / AI agent 应用样本。
- 当前可保守迁出：它补充 [[AI 工具化知识工作流]] 中“AI 把表单、简历、平台交互变成自动化流程”的高风险边界样本。

## 当前边界

- 仓库已 archived，不能按活跃项目推荐。
- 自动投递涉及平台条款、招聘公平性、身份真实性、反滥用和个人数据安全，当前不迁出操作教程。
- README 里的媒体报道链接只能说明传播背景，不能证明项目当前可用或合规。
