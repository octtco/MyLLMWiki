# GitHub - anthropic-cookbook

## 一句话总结

Anthropic Cookbook 是 Claude API 的官方配方库，最适合当作“可复制代码片段 + 能力示例”的工程入口，而不是当成抽象 prompt 清单。

## 当前范围

本页只整理仓库 README 入口信息，不逐个 notebook 拆解，也不验证示例代码是否仍全部可运行。

## 摘要

- README 把 cookbook 定位为帮助开发者构建 Claude 应用的 code snippets 和 guides。
- 内容按能力分组，包括 tool use / integration、third-party integrations、multimodal capabilities、advanced techniques 等。
- 仓库更像“官方做法样本库”，不是完整 agent 框架，也不是课程。
- 对当前知识库来说，它可补 [[AI 工具化知识工作流]]：知识工作和应用开发不是只靠 prompt，还需要能被复制、迁移、组合的工程配方。

## 证据或原文线索

- GitHub API 抓取时显示仓库未归档，MIT license，stars 约 43.7k。
- README 标题为 `Claude Cookbooks`，并建议新手先看 Claude API Fundamentals course。

## 当前可支持的判断

- 当前来源支持：官方 cookbook 可以作为 Claude API 能力边界和常见集成方式的样本池。
- 当前可保守迁出：在知识库里处理 API 能力时，应区分“概念解释”和“可运行配方”。

## 当前边界

- 本页不支持把 cookbook 里的某个示例直接上升成最佳实践。
- 没有逐个执行 notebook，所以不判断示例代码当前运行状态。
