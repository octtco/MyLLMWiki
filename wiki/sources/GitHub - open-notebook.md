# GitHub - open-notebook

## 一句话总结

open-notebook 是一个开源、隐私优先的 Notebook LM 替代品，面向多模态资料组织、AI 对话、检索、播客生成和自托管知识工作。

## 当前范围

本页只整理 README 和 GitHub API 元数据；未部署 Docker / SurrealDB / 前端，也未导入 PDF、音频、视频或网页测试 RAG 与播客生成。

## 摘要

- README 把 Open Notebook 定位为 privacy-focused alternative to Google's Notebook LM。
- 核心能力包括自托管、18+ AI providers、多模态内容组织、professional podcasts、全文/向量搜索、基于研究材料的对话和多语言 UI。
- README 对比 Google Notebook LM，强调数据主权、AI provider 选择、API access、部署可控和可定制。
- GitHub API 抓取时显示 TypeScript 仓库，MIT license，stars 约 27314，仓库未归档。

## 当前可支持的判断

- 当前来源支持：open-notebook 是个人 / 团队研究资料工作台和 Notebook LM 替代方向样本。
- 当前可保守迁出：它补充 [[AI 工具化知识工作流]] 中“资料摄取、检索、对话、播客和自托管”这一类知识工作应用形态。

## 当前边界

- README 的隐私、部署、播客生成、模型兼容和引用质量口径未复测。
- 自托管不等于自动安全；后续要复核数据库、API key、文件权限和公网暴露边界。
- 当前不判断它相对 Notebook LM 的实际质量优劣。
