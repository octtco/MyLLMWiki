# OpenAI - Work with Codex from anywhere

## 一句话总结

这份官方更新页最稳的价值，是把 Codex 明确推进到“远程持续执行的 coding agent”这条线上，而不再只是桌面前的一次性协作工具。

## 当前范围

本页只整理 OpenAI 这篇官方产品更新页，不补 Codex 其他页面、第三方报道或独立体验。

## 摘要

- 页面核心能力是：在 ChatGPT 移动端里查看和继续派发 Codex 正在执行的任务。
- 官方把使用场景明确写成“人离开桌面后，Codex 继续在远端环境里工作”。
- 页面给出明确使用规模：每周使用 Codex 的人数已超过 400 万。
- 这份来源最值得保留的，不是“最强”之类宣传词，而是它把 coding agent 的工作面扩到远程持续执行。

## 证据或原文线索

- 官方页标题：`Work with Codex from anywhere`
- 页面正文直接写到可以在 `ChatGPT mobile app` 里继续跟进 Codex 任务。
- 页面给出明确数字：`more than 4 million people now use Codex every week`

## 当前可支持的判断

- 当前来源支持：Codex 已经被 OpenAI 推向“远程查看、远程派活、持续执行”的产品形态。
- 当前可保守迁出：AI coding framework 的工作面，正在从本地代码生成扩到跨设备任务接力。

## 当前边界

- 这页不能单独支持“Codex 已经能稳定替代通用办公流程”。
- 使用规模不等于框架质量，也不等于跨场景交付能力已经被验证。
