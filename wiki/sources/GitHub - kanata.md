# GitHub - kanata

## 一句话总结

kanata 是一个跨 Linux、macOS 和 Windows 的软件键盘重映射工具，支持多层按键、tap-hold、宏和 Unicode 等高级输入定制。

## 当前范围

本页只整理 README 和 GitHub API 元数据；未安装 kanata，未加载配置，也未测试各平台权限、驱动或后台运行方式。

## 摘要

- README 把 kanata 定位为 cross-platform software keyboard remapper。
- 核心能力包括多层按键功能、高级按键行为定制、tap-hold、宏和 Unicode。
- README 提供预构建二进制、Rust 构建说明、配置指南、平台已知问题和在线模拟器入口。
- GitHub API 抓取时显示 Rust 仓库，LGPL-3.0 license，stars 约 7449，仓库未归档。

## 当前可支持的判断

- 当前来源支持：kanata 是本地输入效率和键盘层定制工具，不是 AI 工具。
- 当前只作为开发者 / power user 输入效率工具样本保存；暂不强行挂入现有 AI 主题。

## 当前边界

- 键盘拦截和重映射涉及系统权限，macOS、Linux、Windows 的安全模型不同。
- 当前未验证 sudo / Karabiner driver / 后台运行 / 配置语法等实际安装成本。
- 后续若积累更多输入效率工具，再考虑新建“开发者输入效率工具”主题。
