# GitHub - Machine Learning for Trading

## 一句话总结

Machine Learning for Trading 是《Machine Learning for Algorithmic Trading》第二版配套代码仓库，包含 150+ notebooks，覆盖数据、特征工程、模型、回测和策略评估。

## 当前范围

本页只整理 README 与 GitHub 元数据；不提供投资建议，不迁出具体交易策略或实盘操作步骤。

## 摘要

- README 说明该书用 23 章和附录覆盖数据 sourcing、financial feature engineering、portfolio management、supervised / unsupervised ML、金融文本信号、deep learning、GAN 和 deep reinforcement learning。
- 仓库包含 150+ notebooks，展示市场数据、基本面数据、另类数据、文本和图像数据如何进入机器学习交易流程。
- README 把 ML4T workflow 写成 idea、investment universe、data collection、feature extraction、model tuning、strategy design、backtesting 和 performance evaluation 的循环。
- 它提供 installation、data sources、chapter notebooks 和 book website 入口。

## 证据或原文线索

- GitHub API 抓取时显示 stars 约 18.5k，仓库未归档；API 未返回明确 license。
- GitHub 描述为 “Code for Machine Learning for Algorithmic Trading, 2nd edition.”

## 当前可支持的判断

- 当前来源支持：该仓库是机器学习交易学习和研究代码样本，而不是投资系统。
