# GitHub - ChinaTextbook

## 一句话总结

ChinaTextbook 是一个集中收集中国小初高和大学 PDF 教材链接 / 文件的仓库，当前在知识库里只适合先作为待审教育资源入口保存。

## 当前范围

本页只保存 README 和 GitHub 元数据，不下载教材 PDF，不复刻文件列表，不判断其中每份教材的版权状态。

## 摘要

- README 说明项目动机是集中公开教材资源，降低普通人获取教育资料的门槛，并照顾海外华人学习国内教育内容的需求。
- README 当前显式列出数学教材入口，包括小学、初中、高中和大学数学。
- 仓库中部分大 PDF 因 GitHub 文件大小限制被拆分，README 说明了合并工具和下载方式。
- README 还包含重新下载、教材捐献、社区和打赏信息。

## 证据或原文线索

- GitHub API 抓取时显示 license 字段为空，stars 约 71.3k，仓库未归档。
- README heading 包括 `项目的由来`、`学习数学`、`问题：如何合并被拆分的文件？`、`重新下载`、`教材捐献`。
- 仓库描述为“所有小初高、大学PDF教材。”

## 当前可支持的判断

- 当前来源支持：这是一个高关注度教材资源集合入口。
- 当前可保守迁出：它对“开放教育资源如何被民间集中整理”有观察价值，但不自动等于每个文件都适合再分发或长期复用。

## 当前边界

- license 缺失，教材版权和再分发边界必须后续单独复核。
- 当前不把它并入正式学习方法或课程体系主题。
- 不下载或复制仓库里的 PDF 教材文件。

## 关系

- `待验证` -> 教育资源入口：来源值得保留，但版权 / 许可边界未稳定。
