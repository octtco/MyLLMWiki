# GitHub - Tolaria

## 一句话总结

Tolaria 是一个用于管理 Markdown knowledge bases 的桌面应用，定位在本地 markdown 知识库的可视化管理与工作台。

## 当前范围

本页只整理 README 和 GitHub API 元数据；未安装桌面 app，未测试 Homebrew、release 包、本地库打开、安全模型或数据迁移。

## 摘要

- GitHub 描述为 “Desktop app to manage markdown knowledge bases”。
- README 包含 walkthroughs、principles、installation、getting started、tech docs 和 security。
- GitHub API 抓取时显示 TypeScript 仓库，AGPL-3.0 license，未归档。
- GitHub API 抓取时显示 TypeScript 仓库，AGPL-3.0 license，stars 约 13652，仓库未归档。

## 当前可支持的判断

- 当前来源支持：Tolaria 是 markdown-first knowledge base 的桌面管理工具样本。
- 它补充 [[AI 工具化知识工作流]]：知识工作流不只发生在 agent host 内，也需要本地知识库的管理、浏览和编辑界面。
- 对 MyLLMWiki 这类 Obsidian / Markdown 库而言，它是可观察的相邻工具，但当前不等于替代方案。

## 当前边界

- 未测试它对现有 Obsidian vault、frontmatter、wikilink、附件和 Git 工作流的兼容性。
- AGPL-3.0 license 对二次分发和集成有约束，后续改造前要单独看许可。
- 当前不判断它比 Obsidian、Logseq 或自建工具更适合。
