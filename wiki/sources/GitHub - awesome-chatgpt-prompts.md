# GitHub - awesome-chatgpt-prompts

## 一句话总结

awesome-chatgpt-prompts / prompts.chat 是一个大型开放 prompt 库，最适合作为角色 prompt 和社区 prompt 形态样本，不适合直接当作质量证明。

## 当前范围

本页只整理 README 入口，不抓取完整 prompt 数据集，不评价每条 prompt 的效果。

## 摘要

- README 把项目称为 open-source prompt library，覆盖 ChatGPT、Claude、Gemini、Llama、Mistral 等模型。
- 项目从 `Awesome ChatGPT Prompts` 演变为 `prompts.chat`，还提供 self-hosting、CLI、Claude Code plugin、MCP server 等入口。
- 同一生态下的 [[The Prompting Book]] 可以作为配套长书来源，帮助区分 prompt 库、提示词设计原则和 skill / agent 边界。
- 它更像 prompt 资料库和分发平台，而不是单个稳定工作流。
- 对 [[AI coding framework]] 来说，它提供的是素材层边界：prompt 库可以补充框架，但不等同于框架。

## 证据或原文线索

- GitHub API 抓取时显示 stars 约 162.7k，license 为 `NOASSERTION`。
- README 标题区域称其为 `The world's largest open-source prompt library for AI`。

## 当前可支持的判断

- 当前来源支持：prompt 可以作为可共享、可收藏、可自托管的知识资产。
- 当前可保守迁出：prompt 库适合进入样本池，但需要和已验证方法、框架执行纪律分开。

## 当前边界

- stars 和规模不等于 prompt 质量。
- 不把它写成 ChatGPT / Claude / Gemini 的通用最佳提示词标准。
