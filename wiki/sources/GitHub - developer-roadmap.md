## 一句话总结

developer-roadmap 是 roadmap.sh 的开源仓库入口，提供面向开发者职业成长的交互式路线图、最佳实践和问题集。

## 当前范围

本页只整理 README 主入口，不逐条核验 roadmap.sh 上每一条路线，也不把路线图里的所有主题拆成概念页。

## 摘要

- 用户给出的 `kamranahmedse/developer-roadmap` 当前 GitHub API 返回 canonical full name 为 `nilbuild/developer-roadmap`，本来源包保留了原请求名和 canonical 名。
- README 把 roadmap.sh 描述为 community driven roadmaps、articles 和 resources。
