# GitHub - Harness

## 一句话总结

Harness Open Source 是一个端到端开发者平台样本，覆盖 SCM、CI/CD pipelines、hosted developer environments 和 artifact registries。

## 当前范围

本页只整理 README 与元数据，不本地运行 Harness，也不比较 Drone、GitHub Actions、GitLab CI 等平台。

## 摘要

- GitHub 描述把 Harness Open Source 定位为 end-to-end developer platform。
- README heading 包括 `Overview`、`Running Harness locally`、`Where is Drone?`、`Harness Open Source Development`。
- 项目关注从源码管理到流水线、开发环境和制品注册表的一体化工程平台。

## 证据或原文线索

- GitHub API 抓取时显示 Apache-2.0 license，stars 约 36.3k，仓库未归档。
- 仓库 homepage 指向 `https://www.harness.io/open-source`。

## 当前可支持的判断

- 当前来源支持：Harness 是开发者平台和 CI/CD 工程工具链方向的开源样本。
- 当前可保守迁出：开发者平台正在把代码、流水线、环境和制品统一到一个工作面。

## 当前边界

- 不判断它相对其他 CI/CD 平台的成熟度。
- 本库当前 DevOps 来源还少，先不新建大主题页。
