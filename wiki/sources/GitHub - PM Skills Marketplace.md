# GitHub - PM Skills Marketplace

## 一句话总结

PM Skills Marketplace 是一组面向产品经理决策的 agentic skills、commands 和 plugins，覆盖 discovery、strategy、execution、launch 和 growth。

## 当前范围

本页只整理 README 和 GitHub API 元数据；未安装 marketplace 或插件，未测试 Claude Cowork、Claude Code、Codex CLI、OpenCode / Gemini CLI 适配。

## 摘要

- README 标题为 “PM Skills Marketplace: The AI Operating System for Better Product Decisions”。
- 它把能力分成 skills、commands、plugins，并提供 Claude Cowork、Claude Code、Codex CLI 和其他 AI assistants 的安装方式。
- README 说明 available plugins，并提到可与 PM Brain 组合。
- GitHub API 抓取时显示 未标出主语言 仓库，MIT license，stars 约 12792，仓库未归档。

## 当前可支持的判断

- 当前来源支持：PM 工作可以被拆成一组可安装 skill / command / plugin。
- 它补充 [[AI 工具化知识工作流]]：产品发现、策略、执行、发布和增长能被流程化封装。
- 它也轻量补充 [[提示词设计]]：PM prompt 不只是单条模板，而可能被打包成带触发方式和工具边界的 skills。

## 当前边界

- README 的“AI Operating System”属于项目定位，未验证真实决策质量。
- PM 决策需要业务语境、用户研究和数据验证，不能只靠 skill 输出。
- 未测试 marketplace 安装、跨 host 兼容和与 PM Brain 的组合行为。
