# GitHub - Prompt Engineering Interactive Tutorial

## 一句话总结

Anthropic 的 prompt-eng-interactive-tutorial 是一套面向 Claude 的交互式提示词工程教程，用 9 章课程和练习覆盖 prompt 结构、清晰指令、角色、数据/指令分离、格式、示例、幻觉控制和复杂 prompt。

## 当前范围

本页只整理 README 和 GitHub API 元数据；未跑 Jupyter Notebook，也未复测课程练习在当前 Claude 模型上的效果。

## 摘要

- README 把课程目标写成掌握好 prompt 的基本结构、识别常见失败模式、理解 Claude 优劣势，以及为常见场景从零构建 prompt。
- 课程分成 beginner、intermediate、advanced 三段，包含 9 章和 appendix。
- 章节覆盖 Basic Prompt Structure、Being Clear and Direct、Assigning Roles、Separating Data from Instructions、Formatting Output、Thinking Step by Step、Using Examples、Avoiding Hallucinations 和 Building Complex Prompts。
- GitHub API 抓取时显示 Jupyter Notebook 仓库，stars 约 36219，仓库未归档。

## 当前可支持的判断

- 当前来源支持：它是官方 prompt 教程和练习材料，不是 prompt 库或 agent 框架。
- 当前可保守迁出：它补充 [[提示词设计]] 中“构件化 -> 格式化 -> 示例化 -> 复杂任务”的教学骨架。
- 当前也能补充 [[AI 工具化知识工作流]]：提示词可以被组织成可练习、可迭代、可教学的知识工作资产。

## 当前边界

- README 仍提到 Claude 3 Haiku / Sonnet / Opus 时代的模型口径，后续要结合当前模型重跑练习。
- 不把课程里的章节顺序写成所有模型和所有任务的唯一最优路径。
- Notebook 练习、Google Sheets 版本和答案钥匙都还未本地验证。
