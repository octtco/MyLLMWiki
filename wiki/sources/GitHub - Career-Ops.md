# GitHub - Career-Ops

## 一句话总结

Career-Ops 是一个基于 Claude Code 的 AI 求职系统，包含 skill modes、Go dashboard、PDF generation 和 batch processing，用于把简历、岗位和求职动作组织成工作流。

## 当前范围

本页只整理 README 和 GitHub API 元数据；未安装项目，未导入 CV，未测试 dashboard、PDF、batch processing、Gemini CLI integration 或平台交互。

## 摘要

- README 将 Career-Ops 描述为 AI-powered job search system built on Claude Code。
- 功能包括 14 skill modes、Go dashboard、PDF generation 和 batch processing。
- Quick Start 要求创建 `cv.md`，并让 Claude 根据个人 CV、目标角色、公司列表和语言偏好定制系统。
- GitHub API 抓取时显示 JavaScript 仓库，MIT license，stars 约 50630，仓库未归档。

## 当前可支持的判断

- 当前来源支持：Career-Ops 是求职流程被封装成 Claude Code / skill 工作流的样本。
- 它补充 [[AI 工具化知识工作流]]：个人职业材料、岗位分析、简历生成和批处理可以被工具化。
- 它也提醒这类工作流会碰到身份真实性、个人数据、平台规则和招聘公平性。

## 当前边界

- 求职自动化涉及个人隐私、雇主平台条款、真实性和公平性；当前不迁出自动投递步骤。
- 简历生成和岗位适配需要人工审核，不能默认批量输出即合适。
- 未测试 PDF、dashboard、batch 行为或与真实招聘平台的边界。
