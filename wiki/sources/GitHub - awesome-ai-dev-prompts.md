# GitHub - awesome-ai-dev-prompts

## 一句话总结

这个仓库更像 AI 软件开发 prompt / role prompt 资料库：它说明工程能力正在被拆成可复用的文本角色和任务模板，但它本身还不是 agent framework。

## 当前范围

本页只整理仓库作为 prompt 集合的来源意义，不逐条评价其中的 prompt。

## 摘要

- 仓库主线是收集 AI 软件开发相关 prompt。
- 它和 [[Skill]] 有相邻关系：二者都在把经验外置成可复用文本，但 prompt library 通常缺少触发条件、工具使用纪律和执行验收。
- 它适合作为 AI coding 生态里的“角色能力模板化”样本。
- 对 [[AI coding framework]] 来说，它更像能力材料库，不是完整框架。

## 证据或原文线索

- GitHub 仓库：`https://github.com/piyushrajyadav/awesome-ai-dev-prompts`
- 仓库以 prompt / role prompt 集合方式组织 AI 开发相关材料。

## 当前可支持的判断

- 当前来源支持：AI coding 生态里存在把工程角色能力整理成 prompt 库的路线。
- 当前可保守迁出：prompt 库可以成为 skill、agent 或 workflow 的素材层，但不能自动等同于这些更完整的执行结构。

## 当前边界

- 不把 prompt 数量写成能力强度。
- 不把这个仓库里的 prompt 直接当作已验证工程方法。
