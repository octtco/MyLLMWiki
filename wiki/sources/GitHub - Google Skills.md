# GitHub - Google Skills

## 一句话总结

google/skills 是 Google 产品和技术相关的 Agent Skills 集合，用于把 Google 生态能力组织成可安装、可复用的 agent skill。

## 当前范围

本页只整理 README 和 GitHub API 元数据；未安装任何 Google skill，未测试认证、API 权限、产品覆盖或跨 host 行为。

## 摘要

- README 标题为 Agent Skills，包含 Installation、Available Skills、Support、Contributing 和 License。
- GitHub 描述为 “Agent Skills for Google products and technologies”。
- GitHub API 抓取时显示 Apache-2.0 license，Python 仓库，未归档。
- GitHub API 抓取时显示 Python 仓库，Apache-2.0 license，stars 约 12500，仓库未归档。

## 当前可支持的判断

- 当前来源支持：大型平台方也在把产品能力包装成 agent skills。
- 它补充 [[AI coding framework]] 与 [[Agent 框架设计]]：skill 已经不只是个人 prompt 包，也可以成为平台生态的能力分发单位。
- 它补充 [[AI 工具化知识工作流]]：Google 产品能力进入 agent 工作流时，需要安装、权限和支持边界。

## 当前边界

- README 简短，具体 skill 的安全、认证和权限边界需要逐个看。
- Google 账户、Workspace 数据和 API 权限涉及隐私与组织合规，不能只按安装成功判断可用。
- 当前只把它作为 platform skill collection 样本，不评价实际覆盖质量。
