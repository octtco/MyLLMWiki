# GitHub - Presenton

## 一句话总结

Presenton 是开源 AI presentation generator 与 API 样本，适合观察知识工作产物怎样从聊天回答走向可部署生成服务。

## 当前范围

本页只整理 README 入口，不部署服务，不评估生成幻灯片质量。

## 摘要

- README 把 Presenton 定位为 Gamma、Beautiful AI、Decktopus 的开源替代。
- 项目强调 self-hosting、模型选择、数据控制，以及通过 API 生成 presentation。
- 它同时提供 Desktop、Cloud、Docker / Windows / macOS / Linux 等入口。
- 对当前知识库来说，它是 [[AI 工具化知识工作流]] 里“从文本生成到正式文档 / slide 产物”的样本。

## 证据或原文线索

- GitHub API 抓取时显示 Apache-2.0，stars 约 6.7k，仓库未归档。
- README heading 包括 `Why Presenton`、`Features`、`Generate Presentation via API`、`Roadmap`。

## 当前可支持的判断

- 当前来源支持：知识工作 AI 工具可以围绕“可交付文档形态”和 API 工作流组织，而不是只停留在聊天界面。
- 当前可保守迁出：自托管和模型选择是这类工具区别于 SaaS 的重要卖点。

## 当前边界

- 不判断它生成内容是否达专业设计质量。
- 不比较它和商业工具的真实优劣。
