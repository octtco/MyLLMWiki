# The Prompting Book 长书拆解工作层：范围与骨架

## 范围

本轮覆盖 `The Prompting Book` 的中英两版 PDF，把它作为一份提示词设计长书来源纳入知识库。英文版用于章节骨架和术语判断，中文版用于中文表述对照。

## 目标

本轮目标不是复述全书，而是建立可追溯的来源包、长书工作层和正式来源页，并把稳定部分接回已有的 prompt 资产化、AI coding framework 与 Skill 讨论。

## 非目标

- 不逐页摘录全书。
- 不把每个章节标题自动升成概念页或方法页。
- 不把书中建议写成跨模型、跨时期都成立的强结论。
- 不替代后续真实任务中的 prompt 效果验证。

## 一句话主线

这本书把 prompt 从“灵感式写法”推进为一套可拆解的沟通结构：先理解模型与有效 prompt 的组成，再用角色、格式、示例、迭代、上下文、链式流程和边界处理，让一次性提问逐渐接近可复用工作流。

## 目录骨架

### Introduction

- Preface
- History
- Introduction

这一组章节先交代 `Awesome ChatGPT Prompts` 到 prompts.chat 社区的来历，也说明新版书的定位：整理哪些做法有效、哪些要避免，以及哪些原则在模型变化中仍然有保留价值。

### Foundations

- Understanding AI Models
- Anatomy of an Effective Prompt
- Core Prompting Principles

这一组章节先把模型理解和 prompt 构件讲清楚。核心不是“神奇咒语”，而是 role、context、task、constraints、format、examples、tone 等要素怎样组合。

### Techniques

- Role-Based Prompting
- Structured Output
- Chain of Thought
- Few-Shot Learning
- Iterative Refinement
- JSON & YAML Prompting

这一组章节把常用技巧拆成可操作单元：用角色收束视角，用输出结构稳定结果，用示例教模式，用迭代修正失败，用 JSON/YAML 把文本接进程序。

### Advanced Strategies

- System Prompts & Personas
- Prompt Chaining
- Handling Edge Cases
- Multimodal Prompting
- Context Engineering
- Agents & Skills

这一组章节把单条 prompt 推向应用层：system prompt、链式流程、异常输入、多模态、上下文工程，以及 agent / skill 的组合关系。

### Best Practices

- Common Pitfalls
- Ethics & Responsible Use
- Prompt Optimization

这一组章节给出维护边界：避免常见失败、考虑责任和安全、在质量、成本、速度、一致性之间做优化。

### Use Cases

- Writing & Content
- Programming & Development
- Education & Learning
- Business & Productivity
- Creative Arts
- Research & Analysis

这一组章节按场景落地，但当前更适合当案例池，不适合直接拆成多个正式案例页。

### Conclusion

- The Future of Prompting

结尾强调 prompt 技巧会随模型变化，但清晰沟通、结构化思考和迭代改进仍有保守可迁出的价值。
