# todo/goals

这里放目标页。

目标页负责回答这些问题：

- 为什么现在要做这件事
- 做到什么算完成
- 当前处在哪个阶段
- 主要卡点是什么
- 下一步准备怎么推进

它和 `todo/tasks/` 的区别：

- `goals/` 负责方向、阶段和控制面
- `tasks/` 负责当前可执行动作

它和 `wiki/topics/` 的区别：

- 主题页讲“围绕某个问题现在知道什么”
- 目标页讲“我现在为什么要做、做到哪了、接下来怎么办”

建议做法：

1. 一个长期或中期目标一页
2. 目标页里挂相关对象、相关知识和下一步任务
3. 真正需要执行的动作继续拆到 `todo/tasks/`
4. 阶段性分析先放 `outputs/`，稳定后再回写到 `wiki/`
