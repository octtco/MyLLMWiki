# Agent Offer

一份面向 后端 转型到 AI Agent  的公开复习资料库。


目的是把分散笔记重组为更适合复习、口述和追问深挖的结构化目录。

## 这份仓库适合谁

- 正在从  后端转向 AI Agent / 大模型应用工程的人

## 如何使用

推荐从这里开始：

1. 先看 [docs/interview_prep/README.md](docs/interview_prep/README.md)
2. 再进入 [00_导航](docs/interview_prep/00_导航/README.md)，读总索引和复习路线图
3. 然后按方向进入具体主题目录，只看对应目录下的 `01_核心问答.md`

如果你只想快速开始：

- 先复习 `01_AI`
- 再过一遍 `02_后端`
- 然后看 `03_系统设计`
- 最后刷 `05_项目表达`

## 推荐复习顺序

### 路线 A：后端转 AI Agent

1. `01_AI`
2. `02_后端`
3. `03_系统设计`
4. `05_项目表达`

### 路线 B：后端面试冲刺

1. `02_后端`
2. `03_系统设计`
3. `04_算法`
4. `05_项目表达`

### 路线 C：项目口述与场景题强化

1. `05_项目表达`
2. 反查 `01_AI / 02_后端 / 03_系统设计`
3. 补齐项目里提到的关键技术点
