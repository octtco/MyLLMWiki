# mattpocock/skills - grill-me

## 来源定位

这是 `mattpocock/skills` 仓库里的 `grill-me` skill 文件。当前整理只保存来源位置和结构性说明，不全文搬运。

## 当前可回证线索

- `grill-me` 是一个极短的 skill。
- 它的核心动作是围绕 plan 或 design 连续追问，直到双方达到 shared understanding。
- 它要求沿着 decision tree 逐枝处理依赖，并且一次只问一个问题。
- 它还要求：如果问题能通过探索代码库回答，就先探索代码库，而不是把问题抛给用户。

## 当前边界

- 这份来源能支持“前置澄清型 skill”的存在和最小形态。
- 它不能单独证明这种 skill 总是优于 brainstorming、plan 或其他 harness。
