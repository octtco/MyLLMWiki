# piyushrajyadav/awesome-ai-dev-prompts

## 来源定位

这是一个面向 AI 软件开发的 prompt / role prompt 集合。当前整理把它作为“工程角色 prompt 库”样本保存。

## 当前可回证线索

- 仓库主线是收集不同软件开发角色、任务和场景下的 AI prompt。
- 它更接近 prompt library，而不是 agent framework。
- 它适合观察 AI coding 生态里“角色能力被文本化、模板化、可复用化”的方向。

## 当前边界

- 本次不逐条整理全部 prompt。
- 不把 prompt 数量等同于真实工程能力。
