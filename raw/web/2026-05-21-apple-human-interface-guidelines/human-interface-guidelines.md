# Human Interface Guidelines

Source URL: https://developer.apple.com/design/human-interface-guidelines

Official data snapshot: `files/human-interface-guidelines.json`

## Abstract

The HIG contains guidance and best practices that can help you design a great experience for any Apple platform.

## Top-Level Topics

- Getting started: Create an app or game that feels at home on every platform you support.
- Foundations: Understand how fundamental design elements help you create rich experiences.
- Patterns: Get design guidance for supporting common user actions, tasks, and experiences.
- Components: Learn how to use and customize system-defined components to give people a familiar and consistent experience.
- Inputs: Learn about the various methods people use to control your app or game and enter data.
- Technologies: Discover the Apple technologies, features, and services you can integrate into your app or game.

## Homepage Principles

- Hierarchy: Establish a clear visual hierarchy where controls and interface elements elevate and distinguish the content beneath them.
- Harmony: Align with the concentric design of the hardware and software to create harmony between interface elements, system experiences, and devices.
- Consistency: Adopt platform conventions to maintain a consistent design that continuously adapts across window sizes and displays.
